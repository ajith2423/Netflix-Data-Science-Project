# Netflix Data Cleaning & Visualization Project
## Overview
This project focuses on cleaning and visualizing Netflix dataset using Python.  
The raw dataset was processed by handling missing values, removing duplicates, and analyzing important insights through visualizations.
## Objectives
Clean raw Netflix dataset
Handle missing values and duplicates
Process and analyze data
Create visual reports using graphs and charts
 Understand trends and insights from Netflix content
## Technologies Used
Python
Pandas
Matplotlib
Seaborn
## Dataset
Dataset Name:
 netflix_titles.csv

Dataset Source:
 Kaggle Netflix Dataset

## visualizations Created
 Movies vs TV Shows Count
 Content Added Per Year
 Top 10 Producing Countries
 Ratings Distribution
 Correlation Heatmap

## Key Insights
 Netflix contains more Movies than TV Shows
 United States has the highest Netflix content production
 Content uploads increased rapidly after 2015
 TV-MA is one of the most common content ratings
